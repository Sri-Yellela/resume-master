# Privacy Policy Hosting

This folder is prepared for GitHub Pages.

Enable GitHub Pages for this repository using:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/docs`

After the change is pushed and Pages finishes deploying, the extension privacy policy should be available at:

`https://sri-yellela.github.io/resume-master/privacy/`

If the extension later moves to its own repository, copy `docs/privacy-policy.md` and `docs/privacy/index.html` together so the policy source and public page remain self-contained.
